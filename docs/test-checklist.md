# Test Checklist

## 阶段 0：初始化

- [ ] `git branch --show-current` 返回 `feature/stage-0-initialization`
- [ ] `git remote -v` 中存在 `origin`
- [ ] 项目根目录存在 `project.godot`
- [ ] Godot 可以识别并打开该项目
- [ ] `scenes/main.tscn` 可以作为主场景加载

## 阶段 2：玩家基础控制与射击

- [ ] 运行项目后可直接进入主场景
- [ ] 使用方向键或 `WASD` 可以控制飞机上下左右移动
- [ ] 按 `Space` 或回车可持续发射子弹
- [ ] 子弹会从玩家前方生成并向上飞行
- [ ] 玩家与子弹都不会立即超出屏幕边界

## 阶段 3：基础敌机与命中判定

- [ ] 运行主场景后可直接看到敌机目标
- [ ] 敌机会在屏幕内持续向下移动
- [ ] 玩家子弹击中敌机后，子弹会被清除
- [ ] 玩家子弹击中敌机后，敌机会被清除
- [ ] 命中时可观察到轻量可见反馈

## 阶段 4：玩家受伤与失败判定

- [ ] 主场景中可看到当前血量文本
- [ ] 玩家与敌机接触后会扣除 1 点血量
- [ ] 玩家掉血后血量文本会立即更新
- [ ] 血量归零后会出现 `FAIL` 失败提示
- [ ] 失败后玩家无法继续移动和射击

## 阶段 5：失败后重开闭环

- [ ] 玩家血量归零后会出现 `Press R to Restart` 提示
- [ ] 失败后按 `R` 可以重新开始当前局
- [ ] 重开后玩家位置恢复到初始位置
- [ ] 重开后玩家血量恢复为初始值
- [ ] 重开后敌机重新回到初始状态，失败提示消失

## 阶段 6：障碍物与躲避压力

- [ ] 运行主场景后可直接看到障碍物
- [ ] 障碍物会在屏幕内持续向下移动
- [ ] 玩家与障碍物接触后会扣除 1 点血量
- [ ] 障碍物加入后，原有敌机命中与失败重开闭环仍然成立
- [ ] 玩家除了敌机外，还需要主动躲避障碍物
