# Rules

## 游戏定位

- 一句话定位：单机 2D 飞机空间大战 / 经典街机闯关，玩家驾驶战机在关卡中移动、射击、躲避障碍与敌机，完成生存和清场目标。

## 玩家输入与基础操作

- 方向输入：上下左右移动战机。
- 射击输入：发射前方攻击。
- 当前 MVP 不包含冲刺、蓄力、技能、切换武器或复杂交互键。

## 单局核心循环

- 进入单个关卡。
- 玩家控制战机移动、射击并规避碰撞风险。
- 敌机与障碍持续对玩家形成压力。
- 玩家通过击毁敌机、保持生存和推进关卡来接近通关目标。
- 达成关卡目标后过关，生命归零则失败。

## 关卡目标

- 基础目标是完成一个可玩的单机关卡。
- 关卡内的直接目标包括：
  - 击毁敌机
  - 躲避障碍物
  - 保持生存直到满足清场或通关条件

## 失败条件

- 玩家飞机与敌机或障碍发生碰撞会受损。
- 玩家血条归零时，本局失败。

## 当前 MVP 范围

- 单机
- 2D 关卡制
- 玩家可进行上下左右移动
- 玩家可进行基础射击
- 存在敌机与障碍物带来的躲避和清场压力
- 存在关卡完成与失败判定

## 不在当前 MVP 内

- 联网
- 复杂剧情
- 商店或升级树
- 多机体切换
- 复杂 Boss 多阶段设计
- 存档
- 排行榜
